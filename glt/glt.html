Requirement:Unity 2020.
  1. Premade assets preparation:
    1. Create a new asset bundle name (as shown in 1.box). And create a folder named your new mod(Recommended) (e.g. Test/HudedTutorial (in the last screenshot shown)).
    2. Create your assets and assign them to the current asset bundle in your editor. Like this:
    3. Copy the code: #if UNITY_EDITOR using System.IO; using UnityEditor; using UnityEngine; public class Packer : EditorWindow { [MenuItem("Window/Pack AssetBundles")] private static void ShowWindow() { GetWindow().Show(); } public string path => "AssetBundles"; BuildAssetBundleOptions options = BuildAssetBundleOptions.None; BuildTarget buildTarget = BuildTarget.StandaloneWindows64; private void OnGUI() { options = (BuildAssetBundleOptions)EditorGUILayout.EnumPopup("BuildAssetBundleOptions:", options); buildTarget = (BuildTarget)EditorGUILayout.EnumPopup("BuildTarget:", buildTarget); if (GUILayout.Button("Build")) { if (!Directory.Exists(path)) { Directory.CreateDirectory(path); } BuildPipeline.BuildAssetBundles(path, options, buildTarget); } } } #endif and paste it into a script file in your editor.
    4. Find the button at the top of the window(Window>Pack AssetBundle). (In 2.box)
    5. Click the build button.
    6. Open folder. ((your project)/AssetBundles)
  2. Asset loading in game
    1. Load asset bundle: AssetBundle.LoadFromFile(string path);.
    2. Load asset: AssetBundle.LoadAsset(string name);.
That's all! Congratulations,you finished the tutorial! More code references.
